Le projet pédagogique 3 :

Manipulation d'un fichier csv:

  1. Utilisation d'une liste de dictionnaires en python (structure de structure combinaison de deux structures de données).
  2. L'exemple : Manipulation des livres selon la catégorie et les évaluations faites par les utilisateurs [0-10]
    • Catégorie représente la classe
    • fusionner les évaluations des utilisateurs avec le fichier csv
    • Les évaluation représente les notes données
    • Classer les livres selon les évaluations
  3. Classement des livres selon leurs dates de publications à partir d'un graphe
  4. On ordonne les notes selon leurs appartenances par rapports aux classes

Travailler ces réflexions pour un bon développement durable
