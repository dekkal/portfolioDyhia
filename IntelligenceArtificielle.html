Le projet pédagogique 1 :

  1. Objectif: Classification d’images provenant de la base ImageNET
    Stack technique: Python, Pytorch, Numpy

    Durant ce projet j'ai crée un simple réseau de neurones convolutif(CNN) avec pytorch pour classer les images en différentes classes
    Tout au long de ce projet je me suis familiarisée avec Pytorch, les CNN, le pooling maximum et j'ai crée un modèle CNN pour la classification des images
    L'ensemble de données que j'ai utilisé et celle provenant de la collection Tiny Images. Le projet est structurée ainsi:

    1. Préparation de l'ensemble de données
    2. Création d'un modèle de base pour la classification d'images
    3. Convolution ,pooling
    4. Modèle CNN pour la classification
    5. Hyperparametres, formation de modèle et évaluation

Travailler ces réflexions pour un bon développement durable
